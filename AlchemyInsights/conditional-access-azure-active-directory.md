# <a name="conditional-access"></a>Betinget tilgang

Betinget tilgang er en funksjon i Azure Active Directory. Med betinget tilgang kan du implementere automatiserte tilgangskontrollbeslutninger for å få tilgang til skyappene som er basert på betingelser. Hvis du vil ha mer informasjon, kan du [gå til Konfigurere Location-Based policyer](https://docs.microsoft.com/azure/active-directory/conditional-access/overview).

Som IT-administrator kan du også kontrollere tilgang til SharePoint og OneDrive ressurser basert på definerte nettverksplasseringer eller enheter du stoler på. Hvis du vil ha mer informasjon, kan du ta en titt på koblingen under.

- [Kontrollere tilgang til SharePoint Online- og OneDrive-data basert på nettverksplassering](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location)

- [Kontrollere tilgang fra ikke-administrerte enheter](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
