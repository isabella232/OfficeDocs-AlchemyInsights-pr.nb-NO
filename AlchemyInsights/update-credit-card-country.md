# <a name="update-credit-card-country-information"></a>Oppdatere landsinformasjon for kredittkort

Du kan ikke oppdatere landsinformasjonen for betalingsmåten. Landet som ble tilordnet betalingen, er landet som ble valgt da organisasjonen ble konfigurert på Microsoft 365.
